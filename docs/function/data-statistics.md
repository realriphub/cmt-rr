# 数据看板

CWD 内置提供评论数据分析和访客数据分析功能。

## 评论数据分析

评论数据分析功能包含：

- 总评论数统计
- 最近 30 天评论数据可视化图表
- 按站点分类查看评论数据

## 访客数据分析

访客数据分析功能包含：

### 概览数据

- 总访问量（PV）统计
- 总页面数统计
- 今日访问量
- 昨日访问量
- 本周访问量
- 上周访问量（用于同比对比）
- 本月访问量
- 上月访问量（用于同比对比）
- 最近 30 天访客数据可视化图表

### 访问明细

- 页面访问明细，支持按 PV 排序或查看最新访问记录
- 点赞页面访问记录

### 数据分析

- 同比趋势：对比本周与上周、本月与上月的访问量变化
- 趋势图表：通过可视化图表直观展示访问数据的变化趋势

## 站点筛选

管理后台顶部导航栏提供了全局站点下拉选择器，用于在多站点之间切换数据视图。

在数据看板页面，所有数据统计都会根据选中的站点进行筛选，确保您能准确查看每个站点的数据情况。
